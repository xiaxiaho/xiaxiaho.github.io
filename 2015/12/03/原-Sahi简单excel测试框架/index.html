[原]Sahi简单excel测试框架

发表于 2015年 12月 3日 | 分类于 Sahi总结
字数统计: 707 | 阅读时长 ≈ 3

虽然sahi pro版本有一个excelframework，但是在sahi os被省掉了，所以自己就写了一个超简单的excel测试框架，各位可以在此基础上做更多的个人定制。

首先要让sahi os可以使用excel api ，可以看[这里](http://download.csdn.net/detail/baidu_29649941/9256781)

这个框架的逻辑就是首先用sahi脚本写好函数，然后将函数名及参数放到excel里，通过读取excel顺序执行写好的函数。这样测试开发写好函数及函数使用说明，就可以交给功能测试。功能测试根据自己的测试用例在excel里组成测试流程进行自动化测试。

excel模板定义如下：



























函数名 参数1 参数2 参数3
add 345 hehe kaka
query one two three
delete four me five

框架代码如下：

<span class="hljs-function"><span class="hljs-keyword">function</span> <span class="hljs-title">excelFramework</span><span class="hljs-params">($excel, $sheet)</span> {</span>
    <span class="hljs-comment">//返回一个数据表的二维数组，不带标题行，也就是第0行</span>
    <span class="hljs-keyword">var</span> $excel = _readExcelFile($excel, $sheet);
    <span class="hljs-comment">//循环读取每一行，将每行的数据组成执行函数，通过eval()执行</span>
    <span class="hljs-keyword">for</span> (<span class="hljs-keyword">var</span> $i = <span class="hljs-number">0</span>; $i &lt; $excel.length; $i++) {
    <span class="hljs-comment">//通过buildFunction方法获取组成的函数</span>
        <span class="hljs-keyword">var</span> $F = buildFunction($excel, $i);
        _log($F);
        <span class="hljs-comment">//执行组成的函数</span>
        <span class="hljs-built_in">eval</span>($F);
    }
}   `</pre>

<pre class="prettyprint">`<span class="hljs-function"><span class="hljs-keyword">function</span> <span class="hljs-title">buildFunction</span><span class="hljs-params">($excel, $rowindex)</span> {</span>
    <span class="hljs-keyword">var</span> $f;
    <span class="hljs-comment">//新建一个数组</span>
    <span class="hljs-keyword">var</span> $ar = <span class="hljs-keyword">new</span> <span class="hljs-built_in">Array</span>();
    <span class="hljs-keyword">for</span> (<span class="hljs-keyword">var</span> $i = <span class="hljs-number">1</span>; $i &lt; $excel[$rowindex].length; $i++) {
        <span class="hljs-keyword">if</span> ($excel[$rowindex][$i] != <span class="hljs-literal">null</span>&amp;&amp;$excel[$rowindex][$i]!=<span class="hljs-string">""</span>) {
            <span class="hljs-keyword">var</span> $arg = <span class="hljs-string">"'"</span> + $excel[$rowindex][$i] + <span class="hljs-string">"'"</span>;
            <span class="hljs-comment">//将函数的参数放到数组里</span>
            $ar.push($arg);
        }
    }
    <span class="hljs-comment">//用逗号分隔把读到的参数组成字符串</span>
    <span class="hljs-keyword">var</span> $args = $ar.join(<span class="hljs-string">","</span>);
    <span class="hljs-comment">//最后把第0列的函数名与参数组合成完整的函数</span>
    $f = $excel[$rowindex][<span class="hljs-number">0</span>] + <span class="hljs-string">"("</span> + $args + <span class="hljs-string">")"</span>;
    <span class="hljs-keyword">return</span> $f;
}`</pre>

在sahi脚本里添加上面两段代码，执行excelFramwork方法，如：

<pre class="prettyprint">`excelFramework(<span class="hljs-string">"readExcelFile.xls"</span>, <span class="hljs-string">"Sheet1"</span>);

这样便会一行一行顺序执行自定义的函数实现简单的自动化。
